## Slim
Application, API 구현을 지원하는 PHP micro framework.
<br>
HTTP 요청을 받아 요청에 맞는 콜백 루틴을 호출하고 HTTP 응답을 반환.
<br>
router 기능을 통해 REST API를 비교적 간단하게 구현할 수 있다.

 ## 설치
 ```
 composer require slim/slim "^3.0"
 ```
